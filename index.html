Sure. Here's a clean recursive implementation:
Python 
def flatten(lst):
    """Recursively flatten nested lists."""
    result = []
    for item in lst:
        if isinstance(item, list):
            result.extend(flatten(item))
        else:
            result.append(item)
    return result

# Example usage
nested = [1, [2, [3, 4]], [5, 6]]
print(flatten(nested))  # [1, 2, 3, 4, 5, 6]
